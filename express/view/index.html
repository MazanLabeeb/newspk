Newspk - API

A basic nodejs based news website (dawn) scraper. You can use this API to fetch latest news in English, Dutch or Urdu language. It uses node-fetch and jsdom dependencies and hence is a very light-weighted package.

Usage
GET | https://newspk.herokuapp.com/api

Query Parameters:

You can pass lang and limit as query parameters to get desired results.

Examples
GET | https://newspk.herokuapp.com/api?lang=urdu&limit=10
GET | https://newspk.herokuapp.com/api?lang=english&limit=1
GET | https://newspk.herokuapp.com/api?lang=english&limit=1
